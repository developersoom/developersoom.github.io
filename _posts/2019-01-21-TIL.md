## **Symbol**
오늘 강의에서 symbol 타입에 대해 배웠다.  
primitive type인데 서로 다른 값을 저장하게 된다고..   

```javascript
const myname = Symbol("my name");
const obj = {
  [myname] : "crong",
  age : 11
}

//이렇게는 접근 안됨
obj.myname //undefined  
obj['myname'] //undefined

//이렇게 접근
obj[myname] //crong

//당연히 이렇게는 가능
obj.myname = "new name";
obj.myname //"new name"

obj[myname] //crong
```
